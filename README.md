Food Ordering System (C++)

A simple console-based Food Ordering System developed using C++.
The project allows users to view a menu, place food orders, and generate a bill.
It demonstrates basic C++ concepts like functions, loops, arrays, and conditional statements.

Technologies Used

C++

GCC / MinGW Compiler

How to Run
g++ food_ordering_system.cpp -o food
./food
